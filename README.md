# Teste Back-end na Teddy Open Banking

## Projeto
  - CRUD de um usuário.
  - Tecnologias: TypeScript, NodeJS, Express MongoDB, Mongoose.

## Iniciando Teste

  #### Clone o repositório.
    ```
    > git clone 
    > yarn install
    ```

  #### Para testes
    ```
    > yarn dev
    ```
